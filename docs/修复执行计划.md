**关联问题：** B4（提交 vs 评测不一致）
**预计时间：** 3-4 小时

#### 必须区分的四个对象

| 对象 | 来源 | 用途 |
|------|------|------|
| `agent_submitted_patch` | Agent 通过 `cat patch.txt` 提交的内容 | Agent 自认为提交了什么 |

@dataclass

    r1_declared_sha: str        # Agent patch SHA
    evaluation_sha: str
    workspace_sha: str
    workspace_snapshot_manifest: dict  # edited_files, status, untracked

**不要求 agent_submitted == evaluation**——但必须记录差异。Harness 使用经过 `evaluation_patch` 过滤后的 Patch，不是 workspace 全量 diff。

### P0-5. Workspace Restore 重写（含 5 项保护）

**关联问题：** B5
**预计时间：** 6-8 小时
**验收条件：** 分支恢复后容器内 `git diff --no-ext-diff` SHA 与 R1 workspace_snapshot SHA 一致

#### 修复方案：NamedTemporaryFile + `docker cp`

```python
def restore_workspace(agent, snapshot: WorkspaceSnapshot, base_commit: str) -> RestoreResult:
    temp_path = None
    try:
        # 保护 1: 验证容器 HEAD
        head = agent.env.execute({"command": "cd /testbed && git rev-parse HEAD"})
        if head["output"].strip() != base_commit:
            return RestoreResult(ok=False, reason="HEAD != base_commit")

        # 保护 2: 清理工作区
        agent.env.execute({"command": f"cd /testbed && git reset --hard {base_commit}"})
        agent.env.execute({"command": "cd /testbed && git clean -fd"})  # 注意 -x 需谨慎

        # 写临时文件
        with tempfile.NamedTemporaryFile(mode="w", suffix=".diff", delete=False) as f:
            f.write(snapshot.tracked_diff)
            temp_path = f.name

        # docker cp
        r1 = subprocess.run(["docker", "cp", temp_path, f"{cid}:/tmp/restore.diff"], ...)
        if r1.returncode != 0:
            return RestoreResult(ok=False, reason="docker cp failed")

        # git apply --check（验证再执行）
        r2 = agent.env.execute({"command": "cd /testbed && git apply --check /tmp/restore.diff"})
        if r2["returncode"] != 0:
            return RestoreResult(ok=False, reason=f"git apply --check: {r2['output']}")

        r3 = agent.env.execute({"command": "cd /testbed && git apply --whitespace=nowarn /tmp/restore.diff"})
        if r3["returncode"] != 0:
            return RestoreResult(ok=False, reason="git apply failed")

        # 保护 4: 验证 SHA（不依赖 sha256sum 管道）
        diff_r = agent.env.execute({"command": f"cd /testbed && git diff --binary --no-ext-diff {base_commit}"})
        if diff_r["returncode"] != 0:
            return RestoreResult(ok=False, reason="git diff failed after restore")
        restored_sha = _sha(diff_r["output"])

        if restored_sha != snapshot.workspace_sha:
            return RestoreResult(ok=False, reason=f"SHA mismatch: {restored_sha} != {snapshot.workspace_sha}")

        return RestoreResult(ok=True, workspace_sha=restored_sha)

    finally:
        # 保护 3: 临时文件清理
        if temp_path and os.path.exists(temp_path):
            os.unlink(temp_path)
```

```python
@dataclass
class RestoreResult:
    ok: bool
    workspace_sha: str = ""
    reason: str = ""

@dataclass
class WorkspaceSnapshot:
    tracked_diff: str          # git diff --binary --no-ext-diff
    untracked_file_list: list  # git ls-files --others --exclude-standard
    state_sha: str             # sha256(tracked_diff)
    base_commit_sha: str       # git rev-parse HEAD
```
